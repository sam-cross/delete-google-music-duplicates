Delete Google Music Duplicates

Duplicate songs in your Play Music library? Use this Python tool to quickly and easily delete duplicate library uploads.

This will delete the tracks with the lowest play counts, instead of deleting the oldest played.

Requires Python 3 and gmusicapi.

Visit the GitHub repository for instructions.
